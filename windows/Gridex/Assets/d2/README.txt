D2 CLI binary placement
=======================

Place d2.exe (Windows x64 build) in this directory.

Download from:
  https://github.com/terrastruct/d2/releases/latest

Steps:
1. Download d2-vX.Y.Z-windows-amd64.zip
2. Extract d2.exe and LICENSE
3. Drop both files into this folder (Assets/d2/)
4. Rebuild the solution

Verify:
  d2.exe --version

Without this binary, the ER Diagram feature will show
"d2.exe not found" when triggered.
